# RegCompassR 1.7.0 condition-pooled architecture

RegCompassR 1.7.0 uses one canonical RNA+ATAC workflow.

```text
condition × cell type cells pooled across biological samples
→ SuperCell2 condition-level metacells
→ condition × cell type Pando GRNs
→ condition-specific GRN meta-modules and local FASTCORE completion
→ one shared union-GEM and one shared medium
→ TF–ATAC regulation integrated into gene support before GPR aggregation
→ directional COMPASS-like minimum-penalty scoring
→ descriptive comparison of condition-level metacells
```

## Multiome gene support

For gene `g` in metacell `u`, RNA support is bounded as

\[
C^{RNA}_{g,u}=\frac{x_{g,u}}{x_{g,u}+h}.
\]

For each Pando TF–peak–gene edge, normalized peak accessibility and TF abundance
are multiplied. Edge activity is centered and scaled across conditions within
the same cell type, then transformed with `tanh`. Pando coefficient magnitudes define
within-gene edge weights, coefficient signs define activation or repression, and
target-model R-squared shrinks unreliable modifiers. The resulting modifier is
bounded:

\[
-1\le R_{g,u}\le 1.
\]

The canonical integration is a zero-preserving update on the support log-odds
scale:

\[
C^{MO}_{g,u}=
\frac{C^{RNA}_{g,u}2^{\alpha R_{g,u}}}
{1-C^{RNA}_{g,u}+C^{RNA}_{g,u}2^{\alpha R_{g,u}}}.
\]

Consequently, absent RNA support remains absent, positive regulation raises
existing support, negative regulation lowers it, and all values remain in
`[0,1]`.

## GPR and reaction expression

Multi-subunit complexes use the Boltzmann minimum-biased mean with `tau = 0.20`.
Isozyme groups are additive. No promiscuity correction is applied:

\[
E_{r,u}=\sum_{k\in isozyme(r)}
BoltzmannMin_{\tau=0.20}\left(C^{MO}_{g,u}:g\in k\right).
\]

## COMPASS-like penalty

Pando is not added as an independent reaction-level penalty. The only
expression-derived LP cost is

\[
p_{r,u}=\frac{1}{1+\log_2(1+E_{r,u})}.
\]

This cost is finite, strictly positive, and monotonically decreases as multiome
reaction expression increases. Structural reaction roles retain fixed curated
costs.

## Shared structural model

Each condition-specific meta-module is completed locally with FASTCORE. The
completed modules are deduplicated to form one union-GEM. All conditions use the
same stoichiometric matrix, reaction bounds, extracellular medium, target set,
and target-flux fraction. Therefore condition differences arise from the
multiome penalty rather than different network structures.

Because biological samples are deliberately mixed before metacell construction,
metacells do not retain one biological-sample identity. The canonical inference
unit is therefore the condition-level metacell, and the package reports
descriptive condition contrasts within each cell type rather than sample-level
significance tests.
