# RegCompassR 1.7.0 condition-pooled architecture

RegCompassR 1.7.0 has one canonical RNA+ATAC workflow:

```text
condition × cell type cells pooled across biological samples
→ SuperCell2 condition-level metacells with original sample composition retained
→ one Pando GRN per condition × cell type
→ complete-GPR core reactions
→ core-reaction subsystem + KEGG/Reactome + master-Rhea reaction expansion
→ local FASTCORE feasibility completion
→ one shared union-GEM and one shared medium
→ RNA support modified by coefficient-weighted ATAC accessibility state
→ directional two-step COMPASS-like LP
→ descriptive condition comparison within cell type
```

## 1. Biological design and pooling

Each biological sample must map to exactly one condition. With
`strict_biological_defaults = TRUE`, every condition must contain at least two
biological samples. Cells are then pooled by `condition × cell type`; sample ID
is deliberately excluded from the SuperCell2 grouping key.

Pooling is cell-count weighted. The workflow therefore retains and writes the
cell-to-metacell membership and, for every metacell:

- the contributing biological samples;
- cell counts and fractions per sample;
- the number of contributing samples;
- the dominant-sample fraction; and
- the effective sample number, `1 / sum(f_s^2)`.

These diagnostics make unequal sample contribution observable. They do not turn
pooled metacells into biological replicates. Condition summaries are descriptive
unless a separate sample-level or externally validated inference layer is used.

## 2. Pando parameters versus per-metacell evidence

Pando coefficients are learned within each `condition × cell type` from the
joint RNA+ATAC data. They describe a fitted regulatory relationship and are not
an independent validation dataset.

For a retained TF–peak–gene edge `e`, the v1.7.0 scoring state uses only the
metacell peak accessibility. Accessibility support is centered and robustly
scaled across conditions within the same cell type, then bounded with `tanh`:

\[
z_{e,u}=\tanh\!\left(\frac{A_{e,u}-\operatorname{median}(A_e)}
{s_e}\right).
\]

For target gene `g`, signed Pando coefficients define normalized edge weights;
model R-squared supplies a reliability shrinkage factor `rho_g`:

\[
R_{g,u}=\rho_g\sum_{e\to g}
\frac{|\beta_e|}{\sum_{k\to g}|\beta_k|}
\operatorname{sign}(\beta_e)z_{e,u},
\qquad -1\le R_{g,u}\le 1.
\]

Metacell TF RNA is not multiplied into this modifier. Target-gene RNA therefore
enters the per-metacell support once, while ATAC supplies the changing regulatory
state. Because the coefficients are still estimated from the same pooled data,
this reduces direct duplicate RNA weighting but does not create fully independent
evidence. External fitting or cross-fitting is required for that stronger claim.

## 3. Multiome gene support

For gene `g` in metacell `u`, full-transcriptome-library-size logCPM is converted
to bounded RNA support:

\[
C^{RNA}_{g,u}=\frac{x_{g,u}}{x_{g,u}+h}.
\]

The regulatory state modifies support on the log-odds scale:

\[
C^{MO}_{g,u}=
\frac{C^{RNA}_{g,u}2^{\alpha R_{g,u}}}
{1-C^{RNA}_{g,u}+C^{RNA}_{g,u}2^{\alpha R_{g,u}}}.
\]

The transform is bounded and zero preserving. ATAC can raise or lower existing
RNA support but cannot create positive support when RNA support is zero.

## 4. GPR aggregation

A reaction is a core reaction only when at least one complete GPR isozyme group
is present in the GRN-derived gene module. A partially represented required
complex is recorded as a candidate but is not treated as core.

For a complex containing `n` required subunits, the canonical AND rule is the
normalized Boltzmann soft minimum:

\[
C_{complex}=-\tau\log\left(\frac1n\sum_{i=1}^{n}
\exp[-C_i/\tau]\right),\qquad \tau=0.20.
\]

Its derivative with respect to every subunit is positive:

\[
\frac{\partial C_{complex}}{\partial C_i}=
\frac{\exp(-C_i/\tau)}{\sum_j\exp(-C_j/\tau)}>0.
\]

Isozyme groups are additive and no gene-promiscuity correction is applied:

\[
E_{r,u}=\sum_{k\in isozyme(r)}C_{complex,k,u}.
\]

`E` is reaction-expression support, not a flux, probability, or kinetic capacity.

## 5. Biological meta-module construction

Annotation-defined biological expansion is ordered:

1. complete-GPR core reactions;
2. all reactions in each core reaction's annotated subsystem;
3. reactions sharing KEGG or Reactome reaction identifiers;
4. reactions sharing the same master Rhea identifier.

Step 4 adds only reactions sharing a master Rhea ID; it does not import every
subsystem containing those reactions. No reaction is included merely because it
shares a metabolite with an included reaction. The architecture has no
metabolite-neighbour, one-hop, stoichiometric-adjacency or currency-metabolite
expansion stage, parameter, membership label, or summary field.

Local FASTCORE is applied only after annotation-defined membership has been
constructed. It may add reactions required to make the target module
flux-consistent. Those additions are labelled separately as FASTCORE feasibility
support and are not reclassified as biological meta-module membership.

## 6. COMPASS-like reaction penalty

Pando is not added as a separate reaction-confidence term. The sole
expression-derived LP cost is

\[
p_{r,u}=\frac{1}{1+\log_2(1+E_{r,u})}.
\]

Missing or zero expression receives cost 1. Curated structural reaction roles
retain explicit fixed costs. The obsolete `P_conf`, Q95 and independent
reaction-confidence interfaces are absent from the canonical penalty API.

This is a RegCompass COMPASS-like transform, not a claim that the expression
penalty is numerically identical to every COMPASS implementation option.

## 7. Shared structural model and two-step LP

All conditions use the same union-GEM, stoichiometric matrix, bounds, medium,
target set, and target-flux fraction. For target reaction `r` and direction
`d in {+1,-1}`, step 1 computes

\[
v^{max}_{r,d}=\max_v d\,v_r
\quad\text{subject to}\quad Sv=0,\;l\le v\le u.
\]

Step 2 computes the minimum evidence-discordance cost:

\[
\min_{v,t}\sum_j p_{j,u}t_j
\]

subject to

\[
Sv=0,\quad l\le v\le u,\quad -t_j\le v_j\le t_j,
\quad d\,v_r\ge \omega v^{max}_{r,d},\quad \omega=0.95.
\]

Thus condition differences arise from the per-metacell penalty matrix, not from
condition-specific network structure or media.

## 8. Output interpretation

The primary output is the minimum LP penalty; lower values indicate stronger
network-wide transcriptomic and accessibility support for sustaining the target
flux. Relative score transforms are secondary and only comparable across units
for the same target and direction. Condition contrasts are descriptive because
biological samples were pooled before metacell construction.
